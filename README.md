🤖 J.A.R.V.I.S. Project (Mark 3)
Добро пожаловать в репозиторий персонального ассистента J.A.R.V.I.S. Этот проект — результат работы энтузиаста, вдохновленного эстетикой киберпанка и современными технологиями.

🛠 О проекте
Frontend: Полностью написан мной с нуля на React/python fast api. Использованы кастомные анимации Framer Motion и стилистика "Neon Glitch".

Backend: серверная часть разработана при активной поддержке ИИ. Код работает стабильно, поэтому вносить изменения в бэкенд крайне не рекомендуется, чтобы не нарушить целостность системы.

3. Тонкая настройка (Prompt)
В файле brain.js вы можете найти константу PROMPT, которая отвечает за манеру речи и поведение Джарвиса.

Предупреждение: Вы можете изменить манеру общения на свой страх и риск, но это крайне не рекомендуется. Настройки промпта тщательно отточены для стабильной работы и корректных ответов.
